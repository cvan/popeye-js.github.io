# popeye.js

Popeye voice controls.


## Installation

To install the [Node](https://nodejs.org/) dependencies:

```sh
npm install
npm run ssl
```


## Local development

To start the local development server:

```sh
npm start
```


## Deployment

To deploy the static pages to [production](https://webvr.rocks/):

```sh
npm run deploy
```
